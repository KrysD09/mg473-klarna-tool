## What This Tool Does

This interactive tool supports Task 3-2A of Individual Assignment 3. It allows you to:

- Set baseline assumptions using **Klarna's real publicly reported figures**
- Select your **functional lens** (Finance, Marketing, Operations, HR, or Innovation)
- Vary a key input across a range to see how the outcome metric changes
- Identify your **tipping point** — the value at which your recommendation flips
- Export or copy a **sensitivity table** for your Appendix

---

## How to Use It

1. **Select your functional lens** using the buttons at the top of the left panel
2. **Review the baseline inputs** — each slider shows Klarna's anchor value and source
3. **Adjust sliders** to reflect your own assumptions
4. **Choose a variable to vary** from the dropdown
5. **Set the variation range** (start with ±30%)
6. **Read the results** — recommendation badge, tipping point box, chart, and table
7. **Copy or print the table** for your Appendix using the buttons below the table

## The Three Alternatives

| Alternative | Description |
|---|---|
| **A — Full AI Automation** | Maximize AI deployment; minimize human agents |
| **B — Structured Hybrid** | AI handles routine inquiries; humans handle complex/sensitive cases |
| **C — Human-First with AI Assist** | Humans remain primary; AI supports behind the scenes |
